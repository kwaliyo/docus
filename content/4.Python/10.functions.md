# Functions in Python

Functions in Python are reusable blocks of code that perform a specific task. They help organize code, reduce repetition, and improve readability. Python supports both built-in functions (e.g., `print()`, `len()`) and user-defined functions, which you can create using the `def` keyword.

## Defining a Function

Functions are defined using the `def` keyword, followed by the function name, parentheses `()` for parameters, and a colon `:`. The code block is indented.

### Basic Syntax

```sh
def function_name(parameters):
    # Code block
    return value  # Optional
```

- function_name: A descriptive name (follows variable naming rules).
- parameters: Optional inputs (can be zero or more).
- return: Optional statement to send a value back (defaults to None if omitted).

Example

```sh
def greet():
    print("Hello, world!")

greet()  # Output: Hello, world!
```

### Parameters and Arguments

Functions can accept parameters to process data.

1. Positional Parameters
   Parameters are specified in order, and arguments must match the order when calling the function.

```sh
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # Output: 8
```

2. Default Parameters
   Parameters can have default values, making them optional when calling the function.

```sh
def greet(name="Guest"):
    print(f"Hello, {name}!")

greet()         # Output: Hello, Guest!
greet("Alice")  # Output: Hello, Alice!
```

3. Keyword Arguments
   Arguments can be passed by parameter name, allowing flexibility in order.

```sh
def introduce(name, age):
    print(f"{name} is {age} years old")

introduce(age=25, name="Bob")  # Output: Bob is 25 years old
```

4. Variable-Length Arguments
   > \*args: Accepts any number of positional arguments as a tuple.

> \*\*kwargs: Accepts any number of keyword arguments as a dictionary.

```sh
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # Output: 10

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="NY")
# Output:
# name: Alice
# age: 25
# city: NY
```

### Return Statement

The return statement exits the function and optionally sends a value back to the caller. Without return, the function returns None.

```sh
def multiply(a, b):
    return a * b

result = multiply(4, 5)
print(result)  # Output: 20

def no_return():
    print("No value returned")

value = no_return()  # Output: No value returned
print(value)         # Output: None
```

### Functions and Operators

Functions often use operators to process data:

- Arithmetic Operators: For calculations (e.g., +, \*).
- Comparison Operators: For conditions (e.g., >, <).
- Logical Operators: For combining conditions (e.g., and, or).

```sh
def is_positive(num):
    return num > 0

print(is_positive(5))   # Output: True
print(is_positive(-3))  # Output: False
```

### Scope and Variables

- Local Scope: Variables defined inside a function are local and not accessible outside.
- Global Scope: Variables defined outside are global; use the global keyword to modify them inside a function.

```sh
x = 10  # Global variable

def modify():
    y = 5  # Local variable
    global x
    x = 20
    print(f"Inside: x={x}, y={y}")

modify()      # Output: Inside: x=20, y=5
print(f"Outside: x={x}")  # Output: Outside: x=20
# print(y)  # Error: y is not defined
```

### Lambda Functions

Lambda functions are small, anonymous functions defined with the lambda keyword. Theyâ€™re useful for short, one-line operations.

Syntax

```sh
lambda arguments: expression
```

Example

```sh
add = lambda x, y: x + y
print(add(3, 4))  # Output: 7

# With map()
numbers = [1, 2, 3]
squared = list(map(lambda x: x**2, numbers))

```

### Common Use Cases

Code Reusability:

```sh
def square(n):
    return n * n

print(square(5))  # Output: 25
print(square(3))  # Output: 9
```

Data Processing:

```sh
def filter_even(numbers):
    return [n for n in numbers if n % 2 == 0]

print(filter_even([1, 2, 3, 4, 5]))  # Output: [2, 4]
```

Modular Programming:

```sh
def calculate_total(price, tax_rate):
    return price + (price * tax_rate)

total = calculate_total(100, 0.1)
print(total)  # Output: 110.0
```

Function Examples with Operators

```sh
# Using arithmetic and comparison operators
def clamp(value, min_val, max_val):
    if value < min_val:
        return min_val
    elif value > max_val:
        return max_val
    return value

print(clamp(15, 0, 10))  # Output: 10
print(clamp(-5, 0, 10))  # Output: 0
print(clamp(5, 0, 10))   # Output: 5
```
