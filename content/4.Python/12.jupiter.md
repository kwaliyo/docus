# Step-by-Step Guide: Setting Up Jupyter Notebook in VS Code for Cybersecurity

This guide provides detailed instructions for setting up Jupyter Notebook in Visual Studio Code (VS Code) to support cybersecurity tasks like data analysis, log parsing, and visualization.

## Overview

- Configure Jupyter Notebook in VS Code for interactive Python development.
- Ideal for cybersecurity tasks such as analyzing logs or threat intelligence data.
- Assumes Python and VS Code are installed (refer to [Setup_Python_VSCode_for_Cybersecurity.md](#) for prerequisites).
- Follow these steps for Windows, macOS, or Linux.

## Step 1: Install Jupyter Extension in VS Code

1. Open VS Code.
2. Go to Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X` on macOS).
3. Search for "Jupyter" (by Microsoft).
4. Click "Install" to add support for Jupyter Notebooks, including notebook creation, execution, and debugging.
5. Verify: Ensure the extension is listed in the Extensions pane.

> **Source**: [VS Code Jupyter Support](https://code.visualstudio.com/docs/datascience/jupyter-notebooks)

## Step 2: Install Jupyter Package

1. Open a terminal in VS Code (`Ctrl+``) or use an external terminal.
2. Ensure your Python environment (preferably a virtual environment) is activated:
   - **Windows**:
     ```bash
     .\.venv\Scripts\activate
     ```
   - **macOS/Linux**:
     ```bash
     source .venv/bin/activate
     ```
3. Install the Jupyter package:
   ```bash
   pip install jupyter
   ```
4. Verify installation:
   ```bash
   jupyter --version
   ```
   - Should display versions for Jupyter components (e.g., `jupyter-core`, `notebook`).

> **Note**: Use a virtual environment to avoid conflicts with system-wide packages.

## Step 3: Configure Python Interpreter for Jupyter

1. Open VS Code and a project folder.
2. Press `Ctrl+Shift+P` to open the Command Palette.
3. Type "Python: Select Interpreter" and choose the Python version in your virtual environment (e.g., `.venv/bin/python` or `.venv\Scripts\python.exe`).
4. Confirm: The selected interpreter should appear in the status bar at the bottom of VS Code.

> **Note**: Ensures Jupyter uses the correct Python environment with installed packages.

## Step 4: Create or Open a Jupyter Notebook

1. In VS Code, create a new file with a `.ipynb` extension (e.g., `cybersecurity_analysis.ipynb`):
   - Press `Ctrl+N` to create a new file, then save it as `filename.ipynb`.
   - Alternatively, use the Command Palette (`Ctrl+Shift+P`), type "Jupyter: Create New Jupyter Notebook," and select it.
2. Verify: The file opens in VS Code’s notebook interface with cells for code and Markdown.
3. Add a sample code cell:
   ```shthon
   print("Jupyter Notebook in VS Code is working!")
   ```
4. Run the cell: Click the "Run Cell" button (play icon) or press `Ctrl+Enter`.

## Step 5: Install Cybersecurity-Relevant Libraries

1. In the activated virtual environment, install libraries useful for cybersecurity:
   ```bash
   pip install pandas numpy matplotlib scapy
   ```
   - `pandas`: Data analysis for logs.
   - `numpy`: Numerical operations.
   - `matplotlib`: Data visualization.
   - `scapy`: Network packet analysis.
2. Verify: In a notebook cell, test an import:
   ```shthon
   import pandas as pd
   print("Pandas imported successfully!")
   ```

> **Source**: [VS Code Python Tutorial](https://code.visualstudio.com/docs/python/python-tutorial)

## Step 6: Test Jupyter with a Cybersecurity Example

1. Create a new notebook (`cybersecurity_test.ipynb`).
2. Add a sample cybersecurity analysis:
   ```shthon
   import pandas as pd
   # Sample log data
   logs = {'IP': ['192.168.1.1', '10.0.0.2'], 'Event': ['Login', 'Failed Login']}
   df = pd.DataFrame(logs)
   print(df)
   ```
3. Run the cell to display the DataFrame.
4. Add a visualization:
   ```shthon
   import matplotlib.pyplot as plt
   df['Event'].value_counts().plot(kind='bar')
   plt.show()
   ```
5. Verify: Output should display a table and a bar chart.

> **Note**: Ensures Jupyter can handle cybersecurity-related data analysis.

## Step 7: Save and Export Notebooks

1. Save your notebook: `Ctrl+S` or File > Save.
2. Export (optional):
   - Use the Command Palette (`Ctrl+Shift+P`), type "Jupyter: Export to Python Script," to convert to `.py`.
   - Alternatively, export to HTML or PDF via File > Export Notebook As.
3. Share with students: Notebooks can be shared as `.ipynb` files for collaboration.

## Teaching Tips for Instructors

- Demonstrate creating and running a notebook live to show cell execution.
- Assign a mini-project: Analyze a sample log file (e.g., CSV) using `pandas` in a notebook.
- Show how to use Markdown cells for documentation (e.g., explaining analysis steps).
- Share resources: [VS Code Jupyter Guide](https://code.visualstudio.com/docs/datascience/jupyter-notebooks).

## Troubleshooting Common Issues

- **Jupyter not found**: Ensure `jupyter` is installed in the active Python environment (`pip install jupyter`).
- **Kernel errors**: Verify the correct Python interpreter is selected; restart VS Code if needed.
- **Library import errors**: Confirm libraries are installed in the virtual environment used by Jupyter.
- Check [VS Code Jupyter Docs](https://code.visualstudio.com/docs/datascience/jupyter-notebooks) for detailed help.
