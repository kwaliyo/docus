# Arrays in Python

In Python, the term "array" typically refers to the built-in `list` data type, which serves as a dynamic, ordered, and mutable collection of items. However, for scenarios requiring a more traditional, fixed-type array (like in C or Java), Python offers the `array` module in its standard library. This section explores both approaches, their properties, and how they interact with operators.

## Lists as Arrays

Python lists are the default choice for array-like functionality due to their flexibility. They can store elements of any data type and grow or shrink dynamically.

### The array Module

For a more traditional array (fixed-type, memory-efficient), Python provides the array module. These arrays are less flexible than lists but optimized for numerical data with a specific type.

### Importing the Module

```py
import array
```

### Creating Arrays

Arrays are created with the array.array(typecode, [initial_values]) constructor, where typecode specifies the data type (e.g., 'i' for integers, 'f' for floats).

```py
# Integer array
int_array = array.array('i', [1, 2, 3, 4])
print(int_array)  # Output: array('i', [1, 2, 3, 4])
print(type(int_array))  # Output: <class 'array.array'>

# Float array
float_array = array.array('f', [1.0, 2.5, 3.7])
print(float_array)  # Output: array('f', [1.0, 2.5, 3.7])
```

### Typecodes

The `array` module uses typecodes to specify the data type of elements in the array. Here are the common typecodes:

| Typecode | C Type        | Python Type | Minimum Size (bytes) |
| -------- | ------------- | ----------- | -------------------- |
| `'b'`    | signed char   | int         | 1                    |
| `'B'`    | unsigned char | int         | 1                    |
| `'i'`    | signed int    | int         | 2                    |
| `'I'`    | unsigned int  | int         | 2                    |
| `'f'`    | float         | float       | 4                    |
| `'d'`    | double        | float       | 8                    |

### Array Operations

Indexing and Slicing

```py
int_array = array.array('i', [10, 20, 30, 40])
print(int_array[0])    # Output: 10
print(int_array[1:3])  # Output: array('i', [20, 30])
```

Modifying Elements

```py
int_array[0] = 100
print(int_array)  # Output: array('i', [100, 20, 30, 40])

int_array.append(50)
print(int_array)  # Output: array('i', [100, 20, 30, 40, 50])
```

Operators with Arrays
Concatenation (+):

```py
arr1 = array.array('i', [1, 2])
arr2 = array.array('i', [3, 4])
combined = arr1 + arr2
print(combined)  # Output: array('i', [1, 2, 3, 4])
```

Repetition (\*):

```py
repeated = array.array('i', [0]) * 3
print(repeated)  # Output: array('i', [0, 0, 0])
```

### Common Array Methods

The `array` module supports many list-like methods. Here are the commonly used ones:

| Method     | Description                             | Example                                                    |
| ---------- | --------------------------------------- | ---------------------------------------------------------- |
| `append()` | Adds an item to the end                 | `array('i', [1, 2]).append(3)` → `array('i', [1, 2, 3])`   |
| `extend()` | Adds items from an iterable             | `array('i', [1]).extend([2, 3])` → `array('i', [1, 2, 3])` |
| `pop()`    | Removes and returns an item             | `array('i', [1, 2]).pop()` → `2`                           |
| `remove()` | Removes the first occurrence of an item | `array('i', [1, 2, 2]).remove(2)` → `array('i', [1, 2])`   |
| `index()`  | Returns the index of an item            | `array('i', [1, 2, 3]).index(2)` → `1`                     |

### Lists vs. `array` Module

| Feature              | List                        | `array` Module                |
| -------------------- | --------------------------- | ----------------------------- |
| **Syntax**           | `[]`                        | `array.array(typecode)`       |
| **Type Flexibility** | Any type                    | Fixed type (e.g., int, float) |
| **Mutability**       | Yes                         | Yes                           |
| **Memory**           | Higher overhead             | More efficient                |
| **Use Case**         | General-purpose, mixed data | Numerical data, performance   |

### Common Use Cases

Dynamic Arrays (Lists):

```py
scores = [85, 90, 88]
scores.append(92)
print(scores)  # Output: [85, 90, 88, 92]
```

Efficient Numeric Arrays (array):

```py
import array
data = array.array('i', [1, 2, 3])
data.extend([4, 5])
print(data)  # Output: array('i', [1, 2, 3, 4, 5])
```

Looping Over Arrays:

```py
for num in array.array('i', [10, 20, 30]):
    print(num)
# Output:
# 10
# 20
# 30
```
